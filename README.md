## üöÄ Getting Started

### 1. Install the Extension
Download and install LexiDrop from the **[Microsoft Edge Add-ons Store](https://microsoftedge.microsoft.com/addons/detail/lexidrop-ai-vocabulary-t/kkidkhomhljchjdhbbggjjchhnnipnji)**.

### 2. Get Your API Key
LexiDrop requires an AI API key to function. We recommend **Google Gemini** for its generous free tier.
- Go to [Google AI Studio](https://makersuite.google.com/app/apikey).
- Create a new API key (it's free).

### 3. Configure LexiDrop
- Click the LexiDrop extension icon in your browser toolbar.
- Go to **Settings** - **Options**.
- Paste your API Key into the field and click **Save**.

### 4. Start Collecting!
You're all set. Highlight a word, right-click, and add it to your drop!
