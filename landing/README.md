# LexiDrop Landing

This app is the marketing site for LexiDrop.

It should present LexiDrop as:

- a browser reading assistant
- translation and explanation directly in the page flow
- focused on Bengali speakers reading Chinese and English content

It should not describe LexiDrop as:

- a vocabulary dashboard
- a physics board
- a general AI experiment

## Local Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Messaging Rules

Keep copy:

- direct
- short
- product-specific
- side-panel-first

Prefer:

- Understand Chinese and English text instantly
- Highlight text and see translation without leaving the page
- Read with translation, explanation, and pronunciation
